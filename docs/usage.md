# **Usage**

Navigate to the repo.

```sh
$ cd PATH_TO_REPO
```


## View available make commands

```bash
$ make help
```


## Format

```sh
$ make fmt-fix
```


## Lint

```sh
$ make lint-fix
```


## Run unit tests

```sh
$ make test
```


## Run

Run application.

```sh
$ make run
```

Start dev server.

```sh
$ make serve
```
