# **Usage**

Navigate to the repo.

```sh
$ cd PATH_TO_REPO
```


## View available make commands

```bash
$ make help
```


## Apply fixes

### Fix all

```sh
$ make fix
```

### Format

Apply formatting fixes.

```sh
$ make fmt-fix
```

### Lint

Apply lint fixes and alert on bad code.

```sh
$ make lint-fix
```


## Test

Run unit tests.

```sh
$ make test
```


## Run

Run application.

```sh
$ make run
```

Start dev server.

```sh
$ make serve
```
